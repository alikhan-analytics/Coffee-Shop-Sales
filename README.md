 ☕ Coffee Shop Sales Analysis

📌 Project Overview

This project analyzes coffee shop sales data to uncover trends in customer purchases, product performance, store performance, and sales behavior. The analysis was conducted using Microsoft Excel, leveraging data cleaning techniques, Pivot Tables, Pivot Charts, and an interactive dashboard.

The goal of this project is to transform raw transactional data into meaningful business insights that can support data-driven decision-making.

---

📊 Dataset Information

Dataset Name: Coffee Shop Sales Dataset
Source: Kaggle
Tool Used: Microsoft Excel
Data Type: Transactional Sales Data

The dataset contains information related to:

* Transaction IDs
* Transaction Dates and Times
* Product Categories
* Product Types
* Store Locations
* Order Quantities
* Sales Revenue
* Product Sizes

---

 🧹 Data Preparation & Cleaning

The dataset was cleaned and prepared before analysis by:

* Checking for missing values

## 🔍 Key Insights

### Sales Trends

* Total sales generated during the analysis period reached **$698,812.33**.
* Sales showed steady growth across the first six months of the year.

### Peak Business Hours

* Customer purchases peaked between **8 AM and 10 AM**, indicating strong morning demand.

### Product Performance

* Coffee products contributed the largest share of sales.
* Barista Espresso emerged as the highest-selling product.

### Store Performance

* Hell's Kitchen generated the highest revenue among all store locations.
* Astoria and Lower Manhattan also contributed significantly to overall sales.

### Customer Behavior

* Friday recorded the highest number of transactions.
* Weekend traffic was comparatively lower than weekday traffic.

### Product Size Preferences

* Regular, Large, and Not Defined sizes accounted for the majority of orders.
